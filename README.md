## System requirements & guide
- PHP 5.5
- PSR-0 Compatible
- PSR-4 Compatible

## Manual include

```php
php composer.phar self-update
php composer.phar install --no-interaction -o

php build/phpunit.phar --configuration ./phpunit/phpunit.xml
```
